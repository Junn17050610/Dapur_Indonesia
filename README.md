# Dapur_Indonesia
Dapur Indonesia — Aplikasi resep masakan Indonesia berbasis web. Fitur: browse resep, detail bahan &amp; cara masak, tambah resep, favorit, dan profil pengguna. Dibangun dengan HTML, CSS, dan JavaScript murni tanpa framework.

# 🍳 Dapur Nusantara

Aplikasi web resep masakan nusantara yang dibangun dengan HTML, CSS, dan JavaScript murni (Vanilla JS) tanpa framework apapun.

## 📄 Halaman
- **Landing Page** — Halaman perkenalan aplikasi
- **Login & Register** — Autentikasi dummy berbasis localStorage
- **Home** — Daftar resep dengan fitur search, filter kategori, dan sort
- **See Recipe** — Detail resep: bahan, langkah memasak, tips, ulasan, dan timer memasak
- **Add Recipe** — Form untuk menambah atau mengedit resep sendiri
- **My Recipe** — Koleksi resep yang diupload dan difavoritkan
- **Profile** — Halaman profil dengan pengaturan akun

## ✨ Fitur
- 🔍 Search & filter resep berdasarkan kategori
- ❤️ Simpan resep favorit
- ➕ Upload & edit resep sendiri
- ⏱️ Timer memasak terintegrasi
- ⭐ Rating & ulasan resep
- 📱 Responsive di semua ukuran layar
- 🖨️ Print resep

## 🛠️ Teknologi
- HTML5 (Semantic)
- CSS3 (Custom Properties, Flexbox, Grid, Animation)
- JavaScript ES6+ (Vanilla)
- localStorage (sebagai penyimpanan data)

## 🚀 Cara Menjalankan
1. Clone repository ini
2. Buka file `landing_page.html` di browser
3. Tidak perlu install apapun
